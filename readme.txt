== Installation ==

= Minimum Requirements =

- PHP 7.4
- WordPress 6.0

= Automatic Installation =

1. Log in to your WordPress admin dashboard.
1. Navigate to the "Plugins" menu.
1. Click on "Add New".
1. Search for "Kepixel".
1. Click on "Install Now" and then on "Activate".

= Manual Installation =

1. Download the plugin.
1. Upload it to your web server using an FTP application.
